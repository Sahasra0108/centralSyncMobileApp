## 🚀 Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Expo Go app (on your Android device) or Android Studio

### Installation

```bash
# Clone the repository
git clone https://github.com/Sahasra0108/centralSyncMobileApp.git

# Navigate to the project directory
cd centralsync-mobile

# Install dependencies
npm install
```

### Run the App

```bash
# Start the Expo development server
npx expo start
```

Then scan the **QR code** with the **Expo Go** app on your Android device. 📷
